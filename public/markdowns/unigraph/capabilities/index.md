# Unigraph Capabilities

This section covers the core capabilities and features of Unigraph.

## Available Capabilities

### Core Graph Features

- **Graph Construction** - Build and manipulate graph structures
- **Graph Layouts** - Visualize graphs with various layout algorithms
- **Graph Filters** - Filter and query graph data
- **Graph Interspection** - Analyze and inspect graph properties

### Development Tools

- **Application Extension** - Extend Unigraph with custom applications
- **App Layout Manager** - Manage application layouts and configurations
- **Document Editing** - Edit and manage documents within Unigraph
- **Project Management** - Organize and manage projects

### Advanced Features

- **Annotation Tooling** - Add annotations and metadata to graph elements
- **Type Theoretic Tooling** - Type-safe graph operations and queries
- **Semantic Web Query Tool** - Query semantic web data and ontologies
- **Copilot Integration** - AI-powered assistance and automation
