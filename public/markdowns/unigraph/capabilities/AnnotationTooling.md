# Annotation Tooling

Add annotations and metadata to graph elements.

## Overview

Annotation tooling provides comprehensive capabilities for adding, managing, and analyzing annotations on graph nodes, edges, and documents.

## Annotation Types

- Text annotations
- Image annotations
- Code annotations
- Link annotations
- Metadata annotations

## Features

- Rich annotation editor
- Annotation search
- Annotation relationships
- Export annotations
- Collaborative annotations
