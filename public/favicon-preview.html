Rounded (6px radius)

To get rounded favicon:

  1. Go to Favicon Generator
  2. Click "Generate Rounded Favicon"
  3. Click "Download Favicon"
  4. Save the file as public/favicon-rounded.png
  5. Update the favicon links in your code

Code to update:

// In index.html
<link rel="icon" type="image/png" href="/favicon-rounded.png" />

// In HtmlPageViewer.tsx
link.href = "/favicon-rounded.png";
